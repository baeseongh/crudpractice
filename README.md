# Note CRUD

title: string, content: text 로 구성된 CRUD 형태의 C9 form 입니다.!
1. git clone
2. Restfull
3. n:1 관계
4. Comment
5. Github push

## RESTful

Route를 더욱 RESTful하게 만들어 봅시다!

## n:1 관계

Comment 모델을 생성하여 Notedhk 1:n 관계를 연결해 봅시다!

## Comment

Comments 컨트롤러를 생성하여 Note show에 댓글 기능을 구현합시다!

## Github push

Github 저장소를 이용하여 완성된 워크스페이스를 올려 봅시다!
